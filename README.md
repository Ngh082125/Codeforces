# My Codeforces Archive

Welcome to my personal repository tracking my competitive programming journey on Codeforces. This space serves as a log of my problem-solving consistency, algorithmic growth, and milestones.

---

## Repository Layout and Workflow

To maintain a fast, completely friction-free workflow, this repository utilizes a strict hybrid organization system designed around contest completion:

* **The Root Folder:** All individual solutions from random problemset hunting or partially completed rounds live directly in the main root directory. Files are named strictly by their unique Problem ID. This specific naming convention allows the system to automatically group files alphabetically, keeping all problems from the same contest neatly lined up next to each other without requiring manual sorting or subfolders.

* **Milestone Folders:** These function as dedicated achievement folders. The exact moment I successfully solve or upsolve **every single problem** from a specific contest, those files are migrated out of the root directory and consolidated into a dedicated milestone folder. This keeps the root folder clean while highlighting fully completed rounds.
